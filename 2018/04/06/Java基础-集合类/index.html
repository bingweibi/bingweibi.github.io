Java基础---集合类

发表于 2018-04-06
名称 安全性 使用场景
ArrayList 线程不安全 快速随机访问（单线程下使用）
LinkedList 线程不安全 快速插入、删除元素
Vector 线程安全 多线程下使用
HashMap 线程不安全 单线程中
HashTable 线程安全 多线程中
ConcurrentHashMap 线程安全 多线程中
TreeMap 线程不安全 单线程，有序

Collection

List

有序队列，每个元素都有索引，从0开始
实现类有LinkedList,ArrayList,Vector,Stack

ArrayList

  1. ArrayList底层是通过一个数组来保存数据的，当构造ArrayList时，如果是默认构造函数的话，默认容量是10
  2. 容量不足时，重新设置容量，新容量 = （原始容量*3）/2+1
  3. 克隆时，全部元素克隆到一个新的数组中
  4. 可以序列化，读取时，先去读取容量在读取元素

遍历方式

  1. 迭代器
  2. 随机遍历
  3. for循环

效率：随机>for>迭代器

LinkedList

  1. 双向链表

Vactor

  1. 底层动态数组，没有指定大小，默认大小10
  2. 容量不足时，若容量增加系数 >0，则将容量的值增加“容量增加系数”；否则，将容量大小增加一倍。
  3. 克隆时，全部元素克隆到一个新的数组中

遍历方式

  1. 迭代器
  2. 随机访问
  3. for循环
  4. Enumeration

效率：随机访问 > Enumeration > for > 迭代器

Stack

  1. 栈，先进后出
  2. 数组实现

Set

不允许有重复元素的集合
实现类有HashSet,TreeSet,HashMap—->HahMap,TreeMap—>TreeMap

HashTree

  1. 没有重复元素的集合
  2. 不保证有序，允许使用null
  3. 不同步

TreeSet

  1. 有序集合
  2. 非同步

Map

  1. 键值对，键不重复，每个键只能映射至多一个值

AbstractMap

HashMap,TreeMap,WeakHashMap

HashMap

  1. 键值对，不保证有序
  2. key,value可以为null
  3. 初始容量（默认16）和加载因子（默认0.75），超过（160.75）、2
  4. “拉链法”解决哈希冲突

TreeMap

  1. 有序键值对(红黑树实现)
  2. 支持序列化

WeakHashmap

HashTable

  1. 键值对，不保证有序
  2. Enumeration遍历
  3. 初始容量（默认11）和加载因子(默认0.75)，超过（110.75）、2+1
  4. “拉链法”解决哈希冲突

Iterator

  1. 通过Iterator迭代器来遍历集合。
  2. 可读取，也可删除

Enumeration

  1. 在Hashtable,Vector,Stack中使用
  2. 只读取，不可修改
